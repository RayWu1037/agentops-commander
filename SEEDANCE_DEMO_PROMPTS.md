# Seedance-Style AI Demo Prompt Pack

Use these prompts with Seedance, Runway, Pika, Kling, Sora-style tools, or another AI video generator.

Important: the Devpost video should still include real screen recording of the hosted app. Use AI-generated clips only as a short intro, transition, or closing visual. The core demo must show the web project functioning.

## Recommended Video Structure

- 0:00-0:08 AI intro clip
- 0:08-2:45 real screen recording of the hosted demo
- 2:45-3:00 AI closing clip or static title card

## Global Style Prompt

```text
Create a polished product-demo visual for an enterprise AI incident response agent called AgentOps Commander. The visual style is clean, modern, operational, and trustworthy: Google Cloud-inspired infrastructure, real-time observability traces, approval gates, and human-supervised automation. Use restrained colors, crisp dashboard lighting, realistic monitors, and no cartoon characters. The mood is confident and production-ready, not futuristic fantasy.
```

## Intro Clip Prompt

```text
8-second cinematic product intro. A modern operations room during a checkout incident. Dashboards show failed orders rising, then an AI agent interface named AgentOps Commander appears. The agent creates a plan, calls tools, and pauses at a clear human approval gate. Subtle Google Cloud-style infrastructure lines connect services. Arize Phoenix-style traces appear as clean timeline spans. Text overlay: "Observable agents that take approved action." Clean enterprise design, realistic UI, no logos except the product name AgentOps Commander, no people close-ups, no copyrighted marks.
```

## Transition Clip Prompt

```text
5-second transition from incident chaos to controlled resolution. Metrics stabilize as a human approval checkpoint turns from Pending to Approved. Tool-call cards and trace spans animate into a clean audit trail. Text overlay: "Plan -> Tools -> Approval -> Trace -> Improve." Modern SaaS product aesthetic, high clarity, restrained motion, no unrelated icons, no ads, no third-party logos.
```

## Closing Clip Prompt

```text
10-second closing visual for AgentOps Commander. Show a clean agent workflow diagram: Goal, Gemini plan, tools, human approval, Arize Phoenix MCP trace review, evaluation scores, next-run improvement. Text overlay: "Human-supervised. Observable. Evaluated." End card includes: "AgentOps Commander" and "Google Cloud Rapid Agent Hackathon". Enterprise dashboard style, readable text, no stock footage, no copyrighted media.
```

## Negative Prompt

```text
Avoid: cartoon mascots, fantasy robots, unreadable UI text, unrelated people, cyberpunk clutter, dark blurry dashboards, third-party logos, copyrighted music references, ads, social media interfaces, fake YouTube pages, medical or legal claims, weapons, private customer data, or any brand marks other than AgentOps Commander.
```

## Editing Notes

- Keep AI clips short so the real product demo remains the majority of the video.
- Do not let AI-generated UI replace the actual hosted app walkthrough.
- Upload `demo-captions.srt` as captions or burn English subtitles into the final video.
- Keep the finished video under 3 minutes.
